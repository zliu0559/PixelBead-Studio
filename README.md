# PixelBead-Studio
网页拼豆图纸生成


一款纯前端、本地运行的图纸生成与排版工具。

1. 克隆或下载本仓库。
2. 双击打开 `index.html` 即可在任意现代浏览器中运行。

### v5.3
* **新增**：移动端原生多点触控支持，实现双指捏合缩放与单指平移。
* **新增**：自动生成图案包边功能，支持自定义边缘颜色，并自动将其映射至当前选中色卡的最接近色号。

### v5.2 & v5.1 & v5.0
* **新增**：响应式布局适配（Responsive Layout），全面支持移动端及平板设备的 UI 交互。

* **新增**：空状态视觉引导与全局 Toast 轻提示反馈系统。
* **修复**：图片缩放抗锯齿边缘（半透明像素）导致的颜色映射错误，引入 Alpha 预混合算法还原真实灰度。
* **修复**：暗黑模式下 Tailwind CSS 样式解析异常问题。
* **修复**：文件上传事件监听丢失的问题。
* **优化**：全量恢复 A-M 系列核心色卡数据，提升边缘色彩匹配的准确度。

### v4.7
* **新增**：智能裁剪的边缘留白控制，支持自定义四周保留 0-8 格空白。
* **新增**：耗材统计列表中加入“总计耗材”置顶显示功能。

### v4.6
* **新增**：包围盒智能裁剪功能，自动去除主要图案四周的无效空白网格，并自适应重置坐标系。

### v4.5
* **新增**：全图防盗水印机制。生成覆盖图纸主体区域的 -30° 倾斜、自适应间距的半透明网纹水印，防止恶意裁剪盗图。

### v4.4
* **重构**：引入原生 Canvas 虚拟摄像机引擎替换 CSS 缩放，实现矢量图级别的无损缩放与鼠标拖拽自由平移。
* **修复**：Canvas 绘制锐角字母（如 M、W 等）时产生的毛刺与越界刺穿问题，统一为圆角连接。

### v4.3
* **新增**：纯色背景智能去除功能，基于泛洪填充算法，自动识别并擦除白底/纯色底，同时实时扣除对应的无效耗材统计。

### v4.2
* **新增**：引入国内主流 A-M 系列色卡的标准 RGB 数据，并设为默认色号标准，彻底解决通过照片取样带来的色偏问题。

### v4.1
* **新增**：品牌色号体系切换模块，支持在 A-M 系列、C 系列与 H 系列间无缝切换，图纸字母及统计同步更新。
* **优化**：强化 5x5 粗线网格坐标系的对齐逻辑。
* **优化**：默认设置专属水印为 "Lzz"，重构左侧控制面板 UI 分组。

### v4.0
* **重构**：全面引入 Tailwind CSS 构建现代化的响应式控制面板与工作区布局。
* **新增**：加入自定义专属图纸水印输入功能，支持在导出图纸中署名。
* **优化**：完善侧边栏耗材清单的动态渲染与按数量降序排列逻辑。

### v3.0
* **新增**：全面支持透明背景 PNG 图片，算法自动识别并跳过 Alpha 通道（透明度）较低的像素，不在透明区域生成错误图纸。
* **新增**：在工作区与导出的图纸中加入 5x5 强化加粗网格与外围数字坐标轴标尺，大幅提升图纸的阅读与数格体验。
* **新增**：视图模式切换功能，支持在“拟真圆珠视图”（圆形）与“方块像素视图”（方形）之间实时切换。

### v2.0
* **新增**：图纸导出功能。利用离屏 Canvas 技术，将生成的图案、水印及耗材清单拼接排版后，导出为单张 PNG 高清海报。
* **新增**：“按住查看原图”功能，支持在转换后的图纸与原始图片之间快速对比。
* **优化**：网格生成算法升级，支持根据原始图片的宽高比，动态自适应计算垂直方向的网格数（图纸高度）。

### v1.0
* **初始化**：PixelBead Studio 项目确立并发布首个原型版本。
* **核心**：实现基础的图片上传与 Canvas 像素数据提取功能。
* **核心**：构建基于欧氏距离的 RGB 色彩匹配算法，实现将任意图片的像素点就近映射为受限的拼豆色卡颜色。
